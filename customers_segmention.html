b) Calculo la matriz de correlacin del dataset

Para poder calcular la matriz de correlacin, primero hay que convertir las columnas categricas a un formato numrico que represente las categoras. Las variables categricas nominales pueden ser convertidas usando tcnicas como one-hot encoding. Las variables categricas ordinales pueden ser asignadas a valores enteros que reflejen su orden y un mtodo comn es el ordinal encoding.


Podemos considerar nominales:

  • Gender: ya que las categoras (Femenino, Masculino) no tienen un orden inherente.
  • CivilState: Nominal, las categoras (Divorciado, Soltero, Casado) no tienen un orden natural.
  • Product: Nominal si representa diferentes tipos o categoras de productos que no tienen un orden natural.

Podemos considerar ordinales:

  • Education: (por ejemplo, Secundaria, Pregrado, Posgrado) se consideran tener un orden donde cada nivel es ms avanzado que el anterior.
  • Salary: las categoras representan rangos de ingresos que tienen un orden natural de menos a ms.

d) qu nos aporta el uso del PCA.

Crees que ser til aplicar una tcnica de reduccin de dimensionalidad a nuestro dataset? Por qu?

Un anlisis PCA nos puede aportar:

  • Reduccin de ruido: Al reducir la dimensionalidad, PCA puede ayudar a eliminar el "ruido" de los datos y resaltar las caractersticas ms importantes.
  • Visualizacin de datos: Con muchas variables, es difcil visualizar los datos y detectar patrones. PCA puede reducir el nmero de variables a 2 o 3 componentes principales que se pueden representar en grficos bidimensionales o tridimensionales.
  • Eficiencia computacional: Menos dimensiones significan menos clculos, lo que puede ser esencial para el anlisis de grandes datasets y puede mejorar la eficiencia de los algoritmos de aprendizaje automtico.
  • Evitar la colinealidad: PCA tambin puede ayudar a evitar problemas de multicolinealidad en los modelos de regresin al crear componentes que son ortogonales (independientes) entre s.
  • Mejor comprensin de las estructuras ocultas: Al enfocarse en las principales fuentes de variacin en los datos, PCA puede revelar estructuras ocultas que no eran aparentes inicialmente.
e) Otra tcnica de reduccin de dimensionalidad es el TSNE. Qu tcnica, para este caso de uso en particular, crees que es la ms recomendable?

  • PCA es una tcnica lineal que reduce la dimensionalidad encontrando los ejes con la mayor varianza, es rpida y adecuada para visualizacin general y para preparar datos para modelos de aprendizaje automtico en datasets grandes.

  • t-SNE es una tcnica no lineal que es excelente para visualizar clusters y patrones complejos en datasets ms pequeos, pero es ms lenta y no tan adecuada para datasets muy grandes o para reducir dimensiones antes de entrenar modelos.

Para este caso, PCA es ms recomendable si se va a modelar predictivamente. t-SNE es mejor si el enfoque est en explorar visualmente los datos y el dataset es de un tamao manejable. Dado que nos han encargado diferentes tcnicas para poder entrenar un modelo predictivo, se conluye que PCA es la mejor opcin en este caso.
